### Added
- Initial VS Code extension scaffold for Matcha
- Matcha language registration for `.mt` and `.matcha` files
- Syntax highlighting via TextMate grammar
- Language configuration for comments and brackets
- Custom Matcha file icons for light and dark themes
- Extension marketplace icon
